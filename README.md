# 関東夏ロボコン2024 自動配膳ロボット制御システム「natsu24」

## 概要
関東夏ロボコン2024の競技テーマ「関東スペシャルランチショー」において、料理の配膳タスクを**完全自動で行う配膳ロボット**の制御システムです。
手動のシェフロボットから受け取った料理の乗ったフライパンを、指定されたテーブルゾーンへ自律的に運搬・配置します。

本大会のルールである**2台のロボット間の通信禁止**という制約下で、シェフロボットとの正確な連携と、ミリ単位での自律航行を実現することを目指しました。本リポジトリは、ROS 2によるシミュレーション環境でのアルゴリズム検証と、マイコンによる低遅延なリアルタイム制御を統合したシステムです。

## システム構成・技術スタック
上位のシミュレーションと下位のハードウェア制御を分割して開発しています。
* **Framework:** ROS 2 (Gazebo、 tf2、 nav_msgs)
* **Languages:** C++、 Python
* **Hardware Control:**
  * モータードライバ: RoboClaw (オムニホイール駆動)
  * センサ: BNO055 IMUモジュール、 VL53L1X ToFセンサ (※本番未使用)、 AMT103 エンコーダ (分解能2048)
  * アクチュエータ: 空圧シリンダ（リフト昇降、アーム開閉用）
* **Algorithms:**
  * 経路追従: Pure Pursuit
  * 制御則: 角度および並進速度のPID制御
  * 自己位置推定: オドメトリとToFセンサを組み合わせた位置補正

## 実装した主要機能
- **ステートマシンによる完全自律シナリオ進行**  
待機状態から、受け渡し地点への移動、フライパンの受取、テーブルゾーンへの配置、そして帰還までの一連の動作を状態遷移（State）として定義し、自律的なミッション遂行を実装しています。

- **オドメトリとToFセンサフュージョンによる自己位置推定 (※コード実装済み、本番運用はオドメトリのみ)**  
エンコーダとIMUから算出したオドメトリ情報に加え、フィールドの壁面との距離をToFセンサで計測し、X/Y座標のズレを動的に補正するクラス (`CorrectPos.h`) を実装しています。

- **Pure PursuitとPIDによる経路追従**  
ロボットの目標位置と現在位置の偏差から、Pure Pursuitアルゴリズムを用いて機体の目標速度・角速度を計算し、足回りのPID制御器を通じてRoboClawへ適切な指令値を送信します。

- **ROS 2 + Gazeboによるシミュレーション環境**  
実機テスト前の安全性確保とアルゴリズム検証のため、xacroによるURDFモデルとGazebo環境を構築し、シミュレーション上で動作確認を行える環境を整備しました。
